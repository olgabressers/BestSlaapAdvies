Technical Reference — affiliate.js

How affiliate link wrapping works

File: /affiliate.js (loaded on all product pages)

How it works: Scans all outbound links on page load, matches hostname against configured retailers, rewrites URL with tracking parameters. MutationObserver handles dynamically added links (compare modals).

No consent gate: Affiliate links wrap for all visitors. Third-party cookies are set by retailers on their own domains.

Opt-out a link: Add data-aff="skip" to the <a> tag.

Debug: Open browser console → type __aff.status()
